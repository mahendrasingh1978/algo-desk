# ALGO-DESK
Automated Options Trading Platform

## Setup
1. Run `bootstrap.sh` on your AWS server
2. Open `installer.html` in your browser
3. Follow the 4-step wizard

## Files
- `bootstrap.sh` — one-time server setup script
- `installer.html` — browser-based control panel
- `docker-compose.yml` — all services
- `backend/` — FastAPI application
- `frontend/` — web dashboard
- `nginx/` — web server config
